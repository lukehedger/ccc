# Configuration

Sulk compiles Solidity based on a set of configuration options:

| Option | Default |
| :------------- | :------------- |
| `contractFileExtension` <string> | `sol` |
| `contracts` <[string]> | None |
| `inputPath` <string> | None |
| `outputFilename` <string> | `"contracts"` |
| `outputPath` <string> | `<inputPath>` |
